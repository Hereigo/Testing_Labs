
### DIRECTIVES ( are Classes that decorated by @Directive ) :

1. Component type Directives :

    -   @Component decorator.


2. Atributive type Directives :

    -   ngModel, ngStyle, ngClass


3. Structural type Directives :

    -   ngFor and ngIf

``
[ngClass]={
    "класс1": true/false,
    "класс2": true/false,
    ...................
}
